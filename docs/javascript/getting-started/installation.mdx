## Package Manager

You can install `@dextr/browser` via npm or yarn:

```bash
# Using npm
npm install @dextr/browser

# Using yarn
yarn add @dextr/browser
```

## CDN

You can also install it from a script tag using the CDN below:

```html
<script
  src="https://browser.dextr.com/0.0.0/bundle.min.js"
  crossorigin="anonymous"
></script>
```
